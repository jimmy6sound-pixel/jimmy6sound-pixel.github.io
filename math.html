一元二次方程

1. 一元二次方程的定义及一般形式

等号两边都是整式，只含有一个未知数(一元)，并且未知数的最高次数是2(二次)的方程，叫做一元二次方程。

ax² + bx + c = 0 (a ≠ 0)

其中ax²是二次项，a是二次项系数；bx是一次项，b是一次项系数；c是常数项。

记忆要点
  • 一元：只含一个未知数
  • 二次：未知数的最高次数是2
  • 整式方程：等号两边都是整式
  • 一般形式：ax²+bx+c=0 (a≠0)
  • 二次项系数a不能为0
例题1：判断一元二次方程

下列关于x的方程中，一元二次方程的个数是（ ）

① ax²+x+2=0 ② 1/x²=3 ③ x=3x² ④ (x+1)(x-1)=x²+4x

解析：

  • ① 当a=0时，不是一元二次方程；当a≠0时，是一元二次方程
  • ② 是分式方程，不是整式方程
  • ③ 整理得3x²-x=0，是一元二次方程
  • ④ 整理可得4x+1=0，是一元一次方程

所以一元二次方程的个数是1或2，取决于a的值。

2. 一元二次方程的解法

解一元二次方程的基本思想是"降次"，具体方法有：

  • 直接开平方法： 适用于形如x²=p或(mx+n)²=p(p≥0)的方程
  • 配方法： 通过配方将方程转化为(mx+n)²=p的形式
  • 公式法： 利用求根公式直接求解
  • 因式分解法： 将方程左边分解为两个一次因式的乘积
记忆要点
  • 直接开平方法：适用于缺少一次项的方程
  • 配方法：关键步骤是"一次项系数一半的平方"
  • 公式法：适用于所有一元二次方程
  • 因式分解法：最简便的方法，但只适用于特殊形式
  • 选择顺序：先考虑因式分解法，再公式法，最后配方法
求根公式：x = [-b ± √(b²-4ac)] / (2a)

3. 根的判别式

一元二次方程ax²+bx+c=0根的判别式：△ = b² - 4ac

  • △ > 0 ⇔ 方程有两个不相等的实数根
  • △ = 0 ⇔ 方程有两个相等的实数根
  • △ < 0 ⇔ 方程无实数根
记忆要点
  • 判别式△=b²-4ac
  • △>0：两个不等实根
  • △=0：两个相等实根
  • △<0：无实根
  • 当a,c异号时，方程一定有两个不相等的实数根

4. 根与系数的关系（韦达定理）

如果方程ax²+bx+c=0(a≠0)的两个实数根为x₁,x₂，那么：

x₁ + x₂ = -b/a, x₁x₂ = c/a
记忆要点
  • 两根之和：x₁+x₂=-b/a
  • 两根之积：x₁x₂=c/a
  • 使用前提：a≠0且△≥0
  • 重要变形：x₁²+x₂²=(x₁+x₂)²-2x₁x₂
  • 重要变形：(x₁-x₂)²=(x₁+x₂)²-4x₁x₂
一元二次方程配套练习

1题目： 若关于x的一元二次方程kx²-2x+3=0有两个实数根，则k的取值范围是______。

解析：

∵ 方程有两个实数根

∴ 需满足两个条件：

  1. k≠0 (二次项系数不为0)
  2. △=(-2)²-4×k×3≥0 → 4-12k≥0 → k≤1/3

∴ k的取值范围是k≤1/3且k≠0。

2题目： 解方程：2x²-5x+2=0

解析：

使用因式分解法：

2x²-5x+2=0

(2x-1)(x-2)=0

∴ 2x-1=0 或 x-2=0

∴ x₁=1/2, x₂=2

3题目： 已知方程x²-3x+1=0，不解方程，求x₁²+x₂²的值。

解析：

根据韦达定理：x₁+x₂=3, x₁x₂=1

x₁²+x₂²=(x₁+x₂)²-2x₁x₂=3²-2×1=9-2=7

4题目： 用配方法解方程：x²+6x-7=0

解析：

x²+6x-7=0

x²+6x=7

x²+6x+9=7+9

(x+3)²=16

x+3=±4

∴ x₁=1, x₂=-7

5题目： 若方程x²+px+q=0的两根为2和-3，求p和q的值。

解析：

根据韦达定理：

2+(-3)=-p → -1=-p → p=1

2×(-3)=q → q=-6

6题目： 某商品原价为200元，连续两次降价后为162元，求平均每次降价的百分率。

解析：

设平均每次降价的百分率为x

200(1-x)²=162

(1-x)²=0.81

1-x=0.9 或 1-x=-0.9

x=0.1=10% 或 x=1.9(舍去)

∴ 平均每次降价的百分率为10%

7题目： 解方程：(x-2)²=9

解析：

使用直接开平方法：

(x-2)²=9

x-2=3 或 x-2=-3

∴ x₁=5, x₂=-1

8题目： 已知关于x的方程x²-2x+m=0有两个相等的实数根，求m的值。

解析：

∵ 方程有两个相等的实数根

∴ △=(-2)²-4×1×m=0

4-4m=0

m=1

9题目： 用公式法解方程：3x²-5x+1=0

解析：

a=3, b=-5, c=1

△=(-5)²-4×3×1=25-12=13

x=[5±√13]/(2×3)=(5±√13)/6

∴ x₁=(5+√13)/6, x₂=(5-√13)/6

10题目： 若方程x²+mx+4=0的一个根是2，求m的值及另一个根。

解析：

将x=2代入方程：2²+2m+4=0 → 4+2m+4=0 → 2m=-8 → m=-4

设另一个根为x₂，根据韦达定理：2+x₂=-m=4 → x₂=2

∴ m=-4，另一个根也是2

11题目： 解方程：x(x-3)=10

解析：

x(x-3)=10

x²-3x-10=0

(x-5)(x+2)=0

∴ x₁=5, x₂=-2

12题目： 若方程x²-4x+k=0的两根之差为2，求k的值。

解析：

设两根为x₁,x₂，且x₁>x₂

根据韦达定理：x₁+x₂=4, x₁x₂=k

又∵ x₁-x₂=2

(x₁-x₂)²=(x₁+x₂)²-4x₁x₂

2²=4²-4k

4=16-4k

4k=12

k=3

二次函数

1. 二次函数的定义

一般地，形如y=ax²+bx+c(a,b,c是常数，a≠0)的函数，叫做二次函数。

其中x是自变量，a,b,c分别是函数解析式的二次项系数、一次项系数和常数项。

记忆要点
  • 二次函数必须是整式函数
  • 自变量的最高次数是2
  • 二次项系数a不能为0
  • 一般形式：y=ax²+bx+c (a≠0)
  • 特殊形式：y=ax², y=ax²+c, y=ax²+bx

2. 二次函数的图象和性质

二次函数y=ax²+bx+c的图象是一条抛物线，性质如下：

性质 a>0 a<0
开口方向 向上 向下
对称轴 x = -b/(2a)
顶点坐标 [-b/(2a), (4ac-b²)/(4a)]
最值 当x=-b/(2a)时，有最小值 当x=-b/(2a)时，有最大值
增减性 在对称轴左侧递减，右侧递增 在对称轴左侧递增，右侧递减
记忆要点
  • 开口方向由a决定：a>0向上，a<0向下
  • 开口大小由|a|决定：|a|越大，开口越小
  • 对称轴公式：x=-b/(2a)
  • 顶点坐标：[-b/(2a), (4ac-b²)/(4a)]
  • 最值：a>0有最小值，a<0有最大值

3. 二次函数与一元二次方程的关系

抛物线y=ax²+bx+c与x轴公共点的横坐标就是方程ax²+bx+c=0的根。

  • 有两个公共点 ⇔ △ > 0
  • 有一个公共点 ⇔ △ = 0
  • 没有公共点 ⇔ △ < 0
记忆要点
  • 二次函数图象与x轴的交点横坐标就是对应一元二次方程的根
  • 交点个数由判别式△决定
  • △>0：两个交点，两个不等实根
  • △=0：一个交点，两个相等实根
  • △<0：没有交点，无实根
二次函数配套练习

13题目： 已知二次函数y=-x²+4x-3。

(1) 用配方法将该二次函数化为y=a(x-h)²+k的形式；

(2) 写出其图象的开口方向、对称轴和顶点坐标。

解析：

(1) y=-x²+4x-3 = -(x²-4x)-3 = -(x²-4x+4-4)-3 = -(x-2)²+4-3 = -(x-2)²+1

(2) ∵ a=-1<0，∴抛物线开口向下

对称轴：x=2

顶点坐标：(2,1)

14题目： 已知二次函数y=2x²-4x+1，求其最小值。

解析：

方法一：配方法

y=2x²-4x+1=2(x²-2x)+1=2(x²-2x+1-1)+1=2(x-1)²-2+1=2(x-1)²-1

∵ a=2>0，∴当x=1时，y有最小值-1

方法二：顶点公式

当x=-b/(2a)=4/(2×2)=1时，y最小值=(4ac-b²)/(4a)=(4×2×1-16)/(4×2)=(8-16)/8=-1

15题目： 已知二次函数y=x²+bx+c的图象经过点(1,0)和(3,0)，求该函数的解析式。

解析：

∵ 图象经过点(1,0)和(3,0)

∴ 设y=a(x-1)(x-3)

又∵ 二次项系数为1，∴a=1

∴ y=(x-1)(x-3)=x²-4x+3

16题目： 求抛物线y=x²-2x-3与x轴的交点坐标。

解析：

令y=0，得x²-2x-3=0

(x-3)(x+1)=0

x₁=3, x₂=-1

∴ 与x轴的交点坐标为(3,0)和(-1,0)

17题目： 已知二次函数y=ax²+bx+c的图象开口向下，对称轴为x=1，且经过点(0,3)，求a,b,c的关系。

解析：

∵ 图象开口向下，∴a<0

∵ 对称轴为x=1，∴-b/(2a)=1 → b=-2a

∵ 经过点(0,3)，∴c=3

∴ a<0, b=-2a, c=3

18题目： 将抛物线y=2x²向右平移3个单位，再向上平移2个单位，求新抛物线的解析式。

解析：

抛物线平移规律：左加右减自变量，上加下减常数项

y=2x²向右平移3个单位：y=2(x-3)²

再向上平移2个单位：y=2(x-3)²+2

∴ 新抛物线的解析式为y=2(x-3)²+2

19题目： 已知二次函数y=x²-4x+3，当x取何值时，y随x的增大而减小？

解析：

y=x²-4x+3=(x-2)²-1

∵ a=1>0，∴抛物线开口向上

在对称轴左侧，y随x的增大而减小

对称轴为x=2

∴ 当x<2时，y随x的增大而减小

20题目： 已知二次函数y=ax²+bx+c的图象如图所示，判断a,b,c的符号。

解析：

从图象可知：

1. 抛物线开口向下，∴a<0

2. 对称轴在y轴右侧，∴-b/(2a)>0，又a<0，∴b>0

3. 抛物线与y轴交于正半轴，∴c>0

∴ a<0, b>0, c>0

21题目： 求二次函数y=-2x²+8x-5的最大值。

解析：

方法一：配方法

y=-2x²+8x-5=-2(x²-4x)-5=-2(x²-4x+4-4)-5=-2(x-2)²+8-5=-2(x-2)²+3

∵ a=-2<0，∴当x=2时，y有最大值3

方法二：顶点公式

当x=-b/(2a)=-8/(2×(-2))=2时，y最大值=(4ac-b²)/(4a)=(4×(-2)×(-5)-64)/(4×(-2))=(40-64)/(-8)=3

22题目： 已知二次函数y=x²-2x-3，求其图象与坐标轴围成的三角形的面积。

解析：

1. 与x轴交点：令y=0，x²-2x-3=0 → (x-3)(x+1)=0 → x₁=3, x₂=-1

∴ 与x轴交点为(3,0)和(-1,0)，两点距离为4

2. 与y轴交点：令x=0，y=-3

∴ 与y轴交点为(0,-3)

3. 三角形面积=1/2×底×高=1/2×4×3=6

23题目： 已知二次函数y=ax²+bx+c的图象经过点(1,2)、(2,3)和(3,6)，求该函数的解析式。

解析：

设y=ax²+bx+c

将三点坐标代入：

a+b+c=2 ①

4a+2b+c=3 ②

9a+3b+c=6 ③

②-①：3a+b=1 ④

③-②：5a+b=3 ⑤

⑤-④：2a=2 → a=1

代入④：3×1+b=1 → b=-2

代入①：1-2+c=2 → c=3

∴ 函数解析式为y=x²-2x+3

24题目： 已知二次函数y=x²-4x+3，当-1≤x≤4时，求y的取值范围。

解析：

y=x²-4x+3=(x-2)²-1

顶点为(2,-1)

当x=-1时，y=(-1)²-4×(-1)+3=1+4+3=8

当x=4时，y=4²-4×4+3=16-16+3=3

∵ 抛物线开口向上，顶点在区间[-1,4]内

∴ 最小值为-1，最大值为8

∴ y的取值范围是-1≤y≤8

旋转

1. 旋转的定义

把一个平面图形绕着平面内某一点O转动一个角度，叫做图形的旋转，点O叫做旋转中心，转动的角叫做旋转角。

旋转三要素：旋转中心、旋转角、旋转方向。

记忆要点
  • 旋转中心：图形绕其旋转的点
  • 旋转角：图形旋转的角度
  • 旋转方向：顺时针或逆时针
  • 三要素缺一不可

2. 旋转的性质

  • 对应点到旋转中心的距离相等
  • 对应点与旋转中心所连线段的夹角等于旋转角
  • 旋转前、后的图形全等
记忆要点
  • 旋转不改变图形的形状和大小
  • 对应点到旋转中心的距离相等
  • 对应点与旋转中心连线的夹角等于旋转角
  • 旋转前后的图形全等

3. 中心对称

把一个图形绕着某一点旋转180°，如果它能够与另一个图形重合，那么就说这两个图形关于这个点对称或中心对称，这个点叫做对称中心。

记忆要点
  • 中心对称是特殊的旋转，旋转角为180°
  • 对称中心是唯一不动点
  • 对称点连线经过对称中心，且被对称中心平分
  • 中心对称的两个图形是全等图形

4. 关于原点对称的点的坐标

两个点关于原点对称时，它们的坐标符号相反，即点P(x,y)关于原点的对称点为P'(-x,-y)。

记忆要点
  • 关于原点对称：横纵坐标都变号
  • 关于x轴对称：横坐标不变，纵坐标变号
  • 关于y轴对称：纵坐标不变，横坐标变号

31题目： 下列选项中的运动形式属于旋转的是（ ）

A. 电梯的升降 B. 钟摆的摆动 C. 抽屉的推拉 D. 篮球的投掷

解析：

A. 电梯的升降：平移

34题目： 如图，将△ABC绕点A顺时针旋转40°得到△ADE，若∠EAC=50°，则∠BAD=______。

圆

1. 圆的基本概念

  • 弦： 连接圆上任意两点的线段
  • 直径： 经过圆心的弦
  • 弧： 圆上任意两点间的部分
  • 等圆： 能够重合的两个圆
  • 等弧： 在同圆或等圆中，能够互相重合的弧
记忆要点
  • 直径是圆中最长的弦
  • 半圆是弧，但弧不一定是半圆
  • 等圆半径相等
  • 等弧只能出现在同圆或等圆中

2. 垂径定理

垂直于弦的直径平分弦，并且平分弦所对的两条弧。

推论： 平分弦(不是直径)的直径垂直于弦，并且平分弦所对的两条弧。

记忆要点
  • 垂径定理：直径⊥弦 → 平分弦、平分弧
  • 推论：直径平分弦(非直径) → 直径⊥弦、平分弧
  • 知二推三：知道其中两个条件可推出其他三个

3. 弧、弦、圆心角的关系

在同圆或等圆中：

  • 相等的圆心角所对的弧相等，所对的弦也相等
  • 相等的弧所对的圆心角相等，所对的弦相等
  • 相等的弦所对的圆心角相等，所对的优弧和劣弧分别相等
记忆要点
  • 前提条件：同圆或等圆
  • 圆心角相等 ↔ 弧相等 ↔ 弦相等
  • 一条弦对应两条弧(优弧和劣弧)

4. 圆周角定理

一条弧所对的圆周角等于它所对的圆心角的一半。

推论：

  • 同弧或等弧所对的圆周角相等
  • 半圆(或直径)所对的圆周角是直角，90°的圆周角所对的弦是直径
记忆要点
  • 圆周角定理：圆周角=1/2圆心角
  • 直径所对的圆周角是90°
  • 90°的圆周角所对的弦是直径
  • 同弧所对的圆周角相等

35题目： 如图，AB是⊙O的直径，弦CD⊥AB，垂足为E，如果AB=10，CD=8，那么AE的长为______。

解析：

连接OC，则OC=OA=OB=5

∵ CD⊥AB，CD=8

∴ CE=DE=4 (垂径定理)

在Rt△OCE中，OE=√(OC²-CE²)=√(25-16)=3

∴ AE=OA-OE=5-3=2

或AE=OA+OE=5+3=8 (当E在OA延长线上时)

∴ AE的长为2或8。

36题目： 在⊙O中，弦AB=8cm，圆心O到AB的距离为3cm，则⊙O的半径为______cm。

解析：

根据垂径定理，过O作OC⊥AB于C，则AC=BC=4cm

在Rt△OAC中，OA=√(OC²+AC²)=√(3²+4²)=√(9+16)=√25=5cm

∴ ⊙O的半径为5cm

37题目： 如图，A、B、C是⊙O上的三点，∠AOB=100°，则∠ACB=______。

解析：

根据圆周角定理，圆周角等于同弧所对圆心角的一半

∠ACB=1/2∠AOB=1/2×100°=50°

A. 相等的圆心角所对的弧相等

B. 平分弦的直径垂直于弦

C. 长度相等的两条弧是等弧

解析：

C. 错误，必须在同圆或等圆中

D. 正确

39题目： 如图，AB是⊙O的直径，C、D是⊙O上的两点，∠BAC=20°，则∠D的度数为______。

解析：

∵ AB是直径，∴∠ACB=90°

∵ ∠BAC=20°，∴∠ABC=70°

∵ ∠D和∠ABC是同弧AC所对的圆周角

∴ ∠D=∠ABC=70°

40题目： 在半径为5cm的⊙O中，弦AB=8cm，则圆心O到弦AB的距离是______cm。

解析：

过O作OC⊥AB于C，则AC=BC=4cm

在Rt△OAC中，OC=√(OA²-AC²)=√(5²-4²)=√(25-16)=√9=3cm

∴ 圆心O到弦AB的距离是3cm

41题目： 如图，四边形ABCD内接于⊙O，∠A=100°，则∠C=______。

解析：

圆内接四边形对角互补

∴ ∠A+∠C=180°

∠C=180°-100°=80°

42题目： 已知⊙O的半径为5，弦AB∥CD，AB=6，CD=8，则AB与CD之间的距离是______。

解析：

分两种情况：

1. 弦AB、CD在圆心同侧：

AB弦心距=√(5²-3²)=√(25-9)=√16=4

CD弦心距=√(5²-4²)=√(25-16)=√9=3

距离=4-3=1

2. 弦AB、CD在圆心异侧：

距离=4+3=7

∴ AB与CD之间的距离是1或7

43题目： 如图，PA、PB是⊙O的切线，A、B为切点，∠APB=40°，则∠AOB=______。

解析：

连接OA、OB

∵ PA、PB是切线，∴OA⊥PA，OB⊥PB

在四边形OAPB中，∠OAP=∠OBP=90°

∴ ∠AOB=360°-90°-90°-40°=140°

44题目： 已知圆锥的底面半径为3cm，母线长为5cm，则圆锥的侧面积为______cm²。

解析：

圆锥侧面积=πrl=π×3×5=15π cm²

45题目： 已知关于x的一元二次方程x²-(2k+1)x+k²+k=0。

(1) 求证：方程有两个不相等的实数根；

(2) 若△ABC的两边AB,AC的长是这个方程的两个实数根，第三边BC的长为5，当△ABC是等腰三角形时，求k的值。

解析：

(1) △=[-(2k+1)]²-4(k²+k)=4k²+4k+1-4k²-4k=1>0

∴ 方程有两个不相等的实数根。

(2) 解方程x²-(2k+1)x+k²+k=0得：

x=[(2k+1)±1]/2，即x₁=k+1, x₂=k

∵ △ABC是等腰三角形，且AB,AC是方程的两根

∴ 有两种情况：

  • 情况1：AB=AC，则k+1=k，无解
  • 情况2：AB=BC=5或AC=BC=5

当AB=k+1=5时，k=4，此时AC=k=4，三边为5,5,4，能构成三角形

当AC=k=5时，AB=k+1=6，三边为5,6,5，能构成三角形

∴ k的值为4或5。

46题目： 如图，抛物线y=ax²+bx+c(a≠0)与x轴交于点A(-1,0)和点B(3,0)，与y轴交于点C(0,3)。

(1) 求抛物线的解析式；

(2) 若点M是抛物线上在x轴下方的一个动点，且△MAB的面积为6，求点M的坐标。

解析：

(1) 设抛物线解析式为y=a(x+1)(x-3)

将C(0,3)代入得：3=a(0+1)(0-3) → 3=-3a → a=-1

∴ 抛物线解析式为y=-(x+1)(x-3)，即y=-x²+2x+3

(2) ∵ A(-1,0), B(3,0)，∴ AB=4

设M(x,y)，则S△MAB=1/2×AB×|y|=1/2×4×|y|=2|y|=6

∴ |y|=3，又∵ M在x轴下方，∴ y=-3

代入抛物线解析式：-3=-x²+2x+3 → x²-2x-6=0

解得：x=1±√7

∴ M点坐标为(1+√7,-3)或(1-√7,-3)

47题目： 如图，在△ABC中，∠ACB=90°，AC=BC，将△ABC绕点C逆时针旋转α角(0°<α<90°)，得到△A₁B₁C，连接BB₁，设CB₁交AB于D，A₁B₁分别交AB、AC于E、F。

(1) 求证：△BCD≌△A₁CF；

(2) 当α等于多少度时，△BB₁D是等腰三角形？

解析：

(1) 证明：

∵ AC=BC，∠ACB=90°，∴ ∠A=∠B=45°

由旋转性质得：CA₁=CA=CB，CB₁=CB=CA，∠A₁CA=∠B₁CB=α

∴ ∠A₁CF=45°+α，∠BCD=45°+α

在△BCD和△A₁CF中：

BC=A₁C，∠BCD=∠A₁CF，∠B=∠A₁=45°

∴ △BCD≌△A₁CF (ASA)

(2) 解：

要使△BB₁D是等腰三角形，分三种情况：

① 当BD=B₁D时，α=30°

② 当BD=BB₁时，α=15°

③ 当B₁D=BB₁时，α=0°(舍去)

∴ 当α=15°或30°时，△BB₁D是等腰三角形

48题目： 如图，AB是⊙O的直径，C是⊙O上一点，D是弧BC的中点，DE⊥AB于E，交BC于F。

(1) 求证：BC=2DE；

(2) 若AB=10，BC=8，求DE的长。

解析：

(1) 证明：

连接OD交BC于G

∵ D是弧BC的中点，∴ OD⊥BC，BG=CG=1/2BC

∵ DE⊥AB，∴ ∠DEO=∠BGO=90°

又∵ ∠DOE=∠BOG，OD=OB

∴ △DOE≌△BOG (AAS)

∴ DE=BG=1/2BC，即BC=2DE

(2) 解：

连接AC，∵ AB是直径，∴ ∠ACB=90°

在Rt△ABC中，AC=√(AB²-BC²)=√(100-64)=√36=6

由(1)知BC=2DE，∴ DE=1/2BC=4

49题目： 某商场销售一批名牌衬衫，平均每天可售出20件，每件盈利40元。为了扩大销售，增加盈利，尽快减少库存，商场决定采取适当的降价措施。经调查发现，如果每件衬衫每降价1元，商场平均每天可多售出2件。

(1) 若商场平均每天要盈利1200元，每件衬衫应降价多少元？

(2) 每件衬衫降价多少元时，商场平均每天盈利最多？

解析：

(1) 设每件衬衫降价x元，则每天可售出(20+2x)件

根据题意：(40-x)(20+2x)=1200

800+80x-20x-2x²=1200

-2x²+60x-400=0

x²-30x+200=0

(x-10)(x-20)=0

x₁=10, x₂=20

∵ 要尽快减少库存，∴ x=20

答：每件衬衫应降价20元

(2) 设每天盈利为y元

y=(40-x)(20+2x)=-2x²+60x+800=-2(x²-30x)+800=-2(x-15)²+1250

∵ -2<0，∴当x=15时，y有最大值1250

答：每件衬衫降价15元时，商场平均每天盈利最多，为1250元

50题目： 如图，在平面直角坐标系中，点A的坐标为(0,2)，点B的坐标为(4,0)，点C在x轴上，且∠ACB=90°。

(1) 求点C的坐标；

(2) 求经过A、B、C三点的抛物线的解析式；

(3) 在抛物线上是否存在点P，使得△PBC是以BC为直角边的直角三角形？若存在，求出所有符合条件的点P的坐标；若不存在，请说明理由。

解析：

(1) 设C(x,0)，由∠ACB=90°得AC²+BC²=AB²

[x²+2²]+[(x-4)²+0²]=4²+2²

x²+4+x²-8x+16=16+4

2x²-8x+20=20

2x²-8x=0

2x(x-4)=0

x₁=0(舍去，与A重合), x₂=4(舍去，与B重合)

∴ 无解？重新考虑：

实际上，∠ACB=90°意味着C在以AB为直径的圆上

AB中点M(2,1)，半径r=1/2AB=1/2√(4²+2²)=√5

圆方程：(x-2)²+(y-1)²=5

令y=0，得(x-2)²+1=5 → (x-2)²=4 → x-2=±2

x₁=0, x₂=4

但x=0时C与A在y轴上，x=4时C与B在x轴上，都不符合∠ACB=90°

∴ 不存在这样的点C？题目可能有误。

假设题目正确，则可能是另一种情况：

设C(x,0)，由AC⊥BC得k_AC·k_BC=-1

(2-0)/(0-x) · (0-0)/(4-x) = -1

左边为0，不可能等于-1

∴ 确实无解

因此，本题可能存在问题，建议检查原题。

中考解题策略总结

  • 仔细审题： 明确已知条件和所求问题，注意单位、取值范围等细节
  • 合理设元： 根据问题特点选择合适的未知数，建立方程或函数模型
  • 数形结合： 充分利用图形帮助理解和解题，特别是几何问题
  • 分类讨论： 当问题存在多种可能情况时，要全面考虑，不重不漏
  • 检验结果： 解出答案后要检验是否符合题意和实际情况
  • 规范书写： 步骤清晰，逻辑严密，书写工整
